Welcome
=======

Welcome to the Access Policy Manager (APM) lab series at Agility |year|.

We will be discussing the two main sections listed below, 

- Zero Trust Application Access (ZTAA): Zero Trust is not just a product to be deployed, but a strategy that spans the whole organization with main three control points Never Trust, Always Verify, Continuously monitoring. 

F5 Identity Aware Proxy provides helps with the building blocks that allows the organization to adopt Zero Trust strategy. 

- API Protection: With F5 Access Policy Manager you can leverage the API protection environment to have authentication and access control at APM along with WAF protection. 


The following labs and exercises will instruct you on how to configure and
troubleshoot various APM use cases based on the experience of solutions 
engineers, support engineers and clients. This guide is intended to
complement lecture material provided during the course as well as a
reference guide that can be referred to after the class as a basis for
configuring Access solutions your own environment.
